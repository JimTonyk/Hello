Ce projet Java permet de créer des objets de type Cercle, Rectangle et Carre et de calculer les surfaces et périmètres à l'aide de méthodes crées dans une interface et redefinies pour chaque objet.

Comment utiliser le programme.
Lancer la commande java -jar Hello.jar dans le dossier téléchargé après avoir téléchargé l'archive depuis GitHub

Contenu du livrable
Fichier readme.txt
Jar avec l'exécutable du projet
Dossiers avec les codes sources
